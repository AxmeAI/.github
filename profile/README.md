# AXME

**Infrastructure and tools for AI agents in production.**

Three products for teams putting AI agents into production — the runtime, the control plane, and the developer tools.

> **Alpha** · [axme.ai](https://axme.ai) · [contact@axme.ai](mailto:contact@axme.ai)

---

## AXME Cloud — durable runtime

Intents API with reliable delivery, async human approval, and long-running workflows that survive restarts. Submit once, track lifecycle, complete later. Replaces polling, webhook glue, and Temporal complexity with one protocol.

**Start here →** [AxmeAI/axme](https://github.com/AxmeAI/axme)
Quickstart, SDK examples, ScenarioBundle, MCP integration, full repo map.

## AXME Mesh — live observability and control

Health dashboard, cost policies, kill switch, and policy enforcement across every agent you ship — yours or third-party. The control plane for agents already running in production.

**Start here →** [AxmeAI/axme](https://github.com/AxmeAI/axme)
Same runtime repo — Mesh features are built into the platform.

## AXME Code — persistent memory for your AI coding agent

MCP plugin for Claude Code that gives your coding agent memory, architectural decisions, and safety guardrails that carry across every session. Open source.
